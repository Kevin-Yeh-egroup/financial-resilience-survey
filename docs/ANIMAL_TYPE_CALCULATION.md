# 狀態理解動物類型計算依據與公式（v4.0）

本文檔說明13種狀態理解動物的計算邏輯、判斷條件和優先順序。

## 一、構面分數計算公式

所有構面分數均為直接加總，範圍 0-30 分：

```typescript
收入穩定度 = 答案[1] + 答案[2]
儲備應變力 = 答案[3] + 答案[5]
債務與保障 = 答案[4] + 答案[6]
金錢管理 = 答案[7] + 答案[8]
資源連結 = 答案[9]
心理與規劃 = 答案[10]
```

**說明：**
- 原始答案範圍：0-10 分
- 構面分數範圍：0-30 分
- 多題構面直接加總
- 單題構面直接使用

---

## 二、分數區間定義（燈號系統 v4.0，全系統統一）

所有六大指標皆以 0–30 分表示，並統一轉換為四段燈號：

- **🔴 0–7**：結構斷裂或高度不足
- **🟠 8–15**：明顯吃力，安全邊際薄
- **🟡 16–23**：可運作但彈性有限
- **🟢 24–30**：結構良好，可承接變動

---

## 三、動物類型判斷邏輯（按優先順序）

### 1. 🐱 縮成一團的貓（高風險疊加型）

**判斷條件（符合以下任一即歸類）：**

**情況一：**
```
六大指標中 🔴 ≥ 2，且 儲備應變力為 🔴
```

**情況二：**
```
儲備應變力為 🔴，且「債務與保障／資源連結／心理與規劃」任一為 🔴
```

**情況三：**
```
「儲備、債務、資源、心理」四項中 🔴 ≥ 3
```

**排他條件：**
- 六大指標中 不得出現任何 🟢

**邏輯說明：**
- 多個關鍵面向同時處於高度脆弱狀態
- 儲蓄、債務、支持、心理四個面向都偏低
- 屬於高風險疊加型狀態

**優勢：**
- 對風險高度敏感，警覺性強
- 在安全情境下，較願意接受陪伴與協助

**風險：**
- 多重結構同時脆弱，突發事件極易引發全面失衡
- 若缺乏即時支持，可能演變為經濟與心理的雙重危機

---

### 2. 🐜 努力拖家的小螞蟻（透支撐持型）

**判斷條件（需同時符合）：**
```
收入穩定度 🟢
儲備應變力 🔴
心理與規劃 ≤ 🟠
資源連結 ≤ 🟠
```

**排他條件：**
- 儲備應變力不得為 🟢
- 若同時符合 🐱 貓，優先歸類為貓

**邏輯說明：**
- 收入穩定度高，能撐住日常運作
- 但儲蓄極低、心理信心低、支持資源低
- 多數責任與壓力集中在自己身上
- 每天都在用力維持，幾乎沒有喘息空間

**優勢：**
- 責任感強，能長時間維持基本運作
- 執行力高，不輕易放棄家庭責任

**風險：**
- 安全邊際極低，任何失業、疾病或意外都可能立即失衡
- 長期心理耗損，容易出現身心俱疲與決策壓力

---

### 3. 🐘 站在細繩上的大象（結構型脆弱）

**判斷條件（需同時符合）：**
```
收入穩定度 🟢
儲備應變力 🟠
心理與規劃 或 資源連結 為 🔴（至少一項）
```

**排他條件：**
- 心理與規劃不得為 🟢
- 資源連結不得為 🟢

**邏輯說明：**
- 收入穩定，但儲備不厚
- 心理壓力大或缺乏支持網絡
- 表面還站得住，其實每一次晃動都需要花很大的力氣才能重新平衡

**優勢：**
- 仍具備穩定收入，短期內可維持基本生活
- 願意承擔責任，不輕易放手

**風險：**
- 抗風險能力不足，一次突發事件就可能導致快速失衡
- 長期心理壓力若未被看見，容易出現結構性崩解

---

### 4. 🐂 努力負重的牛（高負荷撐持型）

**判斷條件（需同時符合）：**
```
收入穩定度 🟢
儲備應變力 🟠
心理與規劃 🟡
資源連結 ≤ 🟠
```

**排他與優先條件：**
- 不得有任何 🔴
- 若心理與規劃 ≤ 🟠，且資源連結 ≤ 🟠，則轉為 🐜 螞蟻
- 若六項皆為 🟠，不歸類本型（轉 🐪 駱駝）

**邏輯說明：**
- 收入仍能支撐生活，心理上也還站得住
- 但儲備不算充足，支援網絡有限
- 靠自己撐很久，長期負荷正在一點一點消耗彈性

**優勢：**
- 穩定度高，責任感與持續力強
- 心理尚具支撐力，能承擔中期壓力

**風險：**
- 支撐點過於集中，缺乏分擔與替代方案
- 若遇到連續性事件（疾病、照顧、工作變動），易出現結構性斷裂

---

### 5. 🐪 撐很久的駱駝（慢性全面吃力型）

**判斷條件（符合以下任一情況即歸類）：**
```
六大指標皆為 🟠
或：五項為 🟠、一項為 🟡
無任何 🔴，但亦未達烏龜條件
```

**排他條件：**
- 不得出現任何 🟢
- 若核心三項（收入、儲備、心理）皆 ≥ 🟡，則轉 🐢 烏龜

**邏輯說明：**
- 沒有立即的危機，也沒有真正的綠洲
- 生活可以運作，但幾乎沒有餘裕
- 所有面向都處在「勉強可以」的狀態
- 長時間慢慢被消耗的狀態

**優勢：**
- 耐力高，能在有限資源下長期維持
- 對壓力與不確定性具有一定適應力

**風險：**
- 長期壓力累積，容易同步耗損心理與財務
- 若缺乏補給與結構轉換，可能在無預警下突然失速

---

### 6. 🦭 被照顧著的水獺（依賴型安全網）

**判斷條件（需同時符合）：**
```
資源連結 🟢
心理與規劃 ≥ 🟡
在「收入穩定度、儲備應變力、金錢管理」三項中至少 2 項 ≤ 🟠
```

**排他與優先條件：**
- 若心理與規劃 🟢 且資源連結 🟢，且結構弱項 ≥ 2，優先判斷為 🐒 猴子
- 不得出現 🔴 ≥ 2（避免與 🐱 貓重疊）

**邏輯說明：**
- 身邊有親友、制度或補助資源，讓生活不至於沉沒
- 當遇到困難時，有人或系統可以接住你
- 但在收入穩定、儲蓄或金錢管理上，仍在建立階段
- 內在的安全感有很大一部分來自外界的照顧

**優勢：**
- 支持網絡完整，不易陷入孤立無援
- 對求助、轉介與使用資源的接受度高

**風險：**
- 若長期停留在被承接狀態，自立結構成長緩慢
- 當支持系統變動時，安全感可能快速下降

---

### 7. 🐒 在樹間移動的猴子（社會韌性型）

**判斷條件（需同時符合）：**
```
資源連結 🟢
心理與規劃 🟢
在「收入穩定度、儲備應變力、金錢管理、債務與保障」四項中 ≤ 🟠 至少 2 項
```

**排他條件：**
- 不得出現任何 🔴
- 若「儲備、債務、管理」皆 ≥ 🟡，則轉 🦅 老鷹（避免低估能力型）

**邏輯說明：**
- 雖然財務結構仍不夠穩定，但你並沒有被困住
- 你知道可以向外連結資源，也願意與人討論、調整方向
- 當環境變動時，你懂得找支點、換路徑，而不是原地硬撐

**優勢：**
- 社會支持與心理彈性強
- 對變動的適應力高，願意嘗試不同解方

**風險：**
- 結構仍偏弱，若支持同時受限，抗風險力不足
- 長期未補強收入、儲備與管理能力，可能形成「靠關係撐」的狀態

---

### 8. 🐿 躲在葉子下的小松鼠（保護網型受傷狀態）

**判斷條件（需同時符合）：**
```
資源連結 🟢
債務與保障 🟢
在以下兩項中至少 1 項為 🟠：儲備應變力、心理與規劃
```

**排他與優先條件：**
- 不得出現任何 🔴
- 若心理與規劃 🟢，且結構弱項 ≥ 2，轉 🐒 猴子
- 若儲備應變力 🟢，且其餘多為 🟡，轉 🐢 烏龜

**邏輯說明：**
- 平時已有制度性保障或支持網絡作為遮蔽
- 只是近期可能遭遇突發事件，讓儲蓄或內心暫時受傷
- 你不是沒有準備，而是正處於修補與恢復的過程中
- 只要給自己時間與適當協助，這層保護網仍能發揮作用

**優勢：**
- 制度性保障與支持系統完整
- 願意使用資源，具備修復條件

**風險：**
- 若長期停留在防禦狀態，行動與結構重建容易延宕
- 過度依賴保護網，忽略自身能力再累積

---

### 9. 🐻 慢慢探出頭的小熊（恢復中狀態）

**判斷條件（需同時符合）：**
```
資源連結 🟢
心理與規劃 🟢
在「收入穩定度、儲備應變力、金錢管理、債務與保障」中：≤ 🟠 至少 1 項，≥ 🟡 至少 1 項
```

**排他條件：**
- 不得符合 🐒 猴子（避免社會韌性型重疊）
- 不得出現任何 🔴

**邏輯說明：**
- 曾經經歷過困難，如今內心已逐漸回暖，也重新看見方向
- 身邊有人支持你，心理上不再那麼害怕
- 但部分財務結構仍在修復中
- 這是一個「心已準備好，結構正在回穩」的過渡階段

**優勢：**
- 心理動能與支持系統穩定
- 已開始實際修復，具備復原能力

**風險：**
- 若修復進度停滯，可能長期停留在過渡狀態
- 需避免過度樂觀而低估結構重建所需時間

---

### 10. 🐶 準備出發的小狗（心理啟動型）

**判斷條件（需同時符合）：**
```
心理與規劃 🟢
儲備應變力 ≤ 🟠
金錢管理 ≤ 🟠
```

**補充條件：**
- 資源連結不要求為 🟢
- 不得出現 🔴 ≥ 2（避免落入 🐱 貓）

**排他條件：**
- 若資源連結 🟢 且心理與規劃 🟢，且結構弱項 ≥ 2 → 轉 🐒 猴子
- 若心理與規劃 🟢，且管理與規劃皆為 🟢 → 轉 🦅 老鷹

**邏輯說明：**
- 已經清楚意識到需要改變，內心出現重新整理與前進的動力
- 對未來也抱有期待
- 只是目前在儲蓄、預算與實際操作方法上，仍缺乏清楚工具與步驟
- 就像準備衝出去的小狗，動能已經出現，但還需要方向與陪伴

**優勢：**
- 改變動機與信心已被啟動
- 對學習新方法與外部協助的接受度高

**風險：**
- 若缺乏具體方法與支持，行動容易停留在想法層次
- 過快嘗試高風險決策，可能增加挫折感

---

### 11. 🦅 盤旋高空的老鷹（高能力、低安全網型）

**判斷條件（需同時符合）：**
```
心理與規劃 🟢
金錢管理 🟢
在以下三項中至少 1 項 ≤ 🟠：儲備應變力、債務與保障、資源連結
```

**排他條件：**
- 不得出現任何 🔴
- 若六大指標多為 🟡 且無明顯弱項 → 轉 🐢 烏龜
- 若綠燈 ≥ 4 且關鍵三項達標 → 轉 🐎 馬

**邏輯說明：**
- 具備良好的判斷與規劃能力，也懂得如何管理金錢與風險
- 你靠的是能力與思考在飛，而不是厚實的安全網
- 一旦遇到強風或突發變化，調整空間仍然有限

**優勢：**
- 分析與規劃能力佳，學習與調整速度快
- 面對問題能主動找解方，而非被動承受

**風險：**
- 過度仰賴個人能力，忽略制度性與外部緩衝
- 一次重大事件，可能造成快速下滑

---

### 12. 🐢 穩定前行的烏龜（日常穩定型，純黃燈層）

**判斷條件（需同時符合以下所有條件）：**
```
六大指標中 🔴 = 0
🟢 = 0（不允許任何綠燈）
🟠 ≤ 1
其餘皆為 🟡
核心三項：收入穩定度 ≥ 🟡、儲備應變力 ≥ 🟡、心理與規劃 ≥ 🟡
```

**排他條件：**
- 若出現任何 🟢 → 不得為烏龜（轉 🐎 或 🦅）
- 若 🟠 ≥ 2 → 轉 🐪 駱駝

**邏輯說明：**
- 生活節奏不快，但結構完整
- 收入、儲蓄、風險準備、金錢管理、支持網絡與心理狀態都維持在安全區間
- 也許沒有太多餘裕，但日常變動不會讓你失去平衡
- 你靠的是持續與耐心，而不是僥倖或硬撐

**優勢：**
- 基本結構完整，不易因小波動失衡
- 心理穩定，對生活具掌控感

**風險：**
- 緩衝厚度有限，仍需逐步累積儲備與支持
- 面對大型風險事件，調節空間仍有限

---

### 13. 🐎 穩健奔跑的馬（成熟韌性型）

**判斷條件（需同時符合以下條件）：**
```
六大指標中 🟢 ≥ 4
在「儲備應變力、資源連結、心理與規劃」三項中至少 2 項為 🟢
無任何 🔴
無任何 🟠
其餘指標至少為 🟡
```

**邏輯說明：**
- 不只站得住，還能持續前行
- 家庭的穩定不是仰賴單一收入或單一關係，而是由多個支柱共同承接
- 即使遇到突發事件，壓力也不會集中在某一個點，而能被分散與消化
- 這代表你的家庭已具備成熟的財務韌性

**優勢：**
- 抗風險能力高，不易因單一事件失衡
- 結構穩定、調節彈性佳，具備長期規劃能力

**風險：**
- 長期穩定可能降低對新風險的警覺
- 需持續檢視保障、資源與環境變化
