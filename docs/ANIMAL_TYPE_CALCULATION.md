# 狀態理解動物類型計算依據與公式

本文檔說明12種狀態理解動物的計算邏輯、判斷條件和優先順序。

## 一、構面分數計算公式

所有構面分數均轉換為 0-100 分制：

```typescript
收入穩定度 = ((答案[1] + 答案[2]) / 2) × 10
儲備應變力 = ((答案[3] + 答案[5]) / 2) × 10
債務與保障 = ((答案[4] + 答案[6]) / 2) × 10
金錢管理 = ((答案[7] + 答案[8]) / 2) × 10
資源連結 = 答案[9] × 10
心理與規劃 = 答案[10] × 10
```

**說明：**
- 原始答案範圍：0-10 分
- 構面分數範圍：0-100 分
- 多題構面取平均值後乘以 10
- 單題構面直接乘以 10

---

## 二、分數區間定義（全系統一致）

所有六大指標與總分皆以 0–100 分表示，分區如下：

- **🔴 0–39**：高度脆弱（紅燈）
- **🟠 40–59**：結構吃力（橘燈）
- **🟡 60–74**：接近穩定（黃燈）
- **🟢 75–100**：韌性良好（綠燈）

---

## 三、動物類型判斷邏輯（按優先順序）

### 1. 🐱 縮成一團休息的貓（高風險疊加型，優先級最高）

**判斷條件（符合以下任一情況）：**

**情況一：**
```
儲備應變力 為 🔴（0–39）
且在「債務與保障、資源連結、心理與規劃」中至少一項為 🔴（0–39）
```

**情況二：**
```
在「儲備應變力、債務與保障、資源連結、心理與規劃」四項中，有三項以上為 🔴（0–39）
```

**邏輯說明：**
- 多個關鍵面向同時處於高度脆弱狀態
- 儲蓄、債務、支持、心理四個面向都偏低
- 屬於高風險疊加型狀態

**優勢：**
- 對自身困境高度有感，對風險的警覺性高
- 一旦獲得適當支持，通常願意配合調整與求助

**風險：**
- 多重壓力疊加，財務與心理脆弱度高
- 若缺乏即時介入與資源連結，容易快速惡化為急難或創傷狀態

---

### 2. 🐜 努力拖著家的小螞蟻（透支撐持型）

**判斷條件（需同時滿足）：**
```
收入穩定度 為 🟢（75–100）
儲備應變力 為 🔴（0–39）
心理與規劃 為 🟠 或 🔴（0–59）
資源連結 為 🟠 或 🔴（0–59）
```

**邏輯說明：**
- 收入穩定度高，能撐住日常運作
- 但儲蓄極低、心理信心低、支持資源低
- 多數責任與壓力集中在自己身上
- 每天都在用力維持，幾乎沒有喘息空間

**優勢：**
- 責任感強、執行力高，能在壓力下維持家庭基本運作
- 對工作與家庭承諾度高，不輕易放棄

**風險：**
- 緩衝極低，任何失業、疾病或意外都可能瞬間失衡
- 長期心理耗損，若未補充支持，容易出現身心俱疲或決策失誤

---

### 3. 🐘 站在細繩上的大象（結構型脆弱）

**判斷條件（需同時滿足）：**
```
收入穩定度 為 🟢（75–100）
儲備應變力 為 🟠（40–59）
且以下任一成立：
  心理與規劃 為 🔴（0–39）
  資源連結 為 🔴（0–39）
```

**邏輯說明：**
- 收入穩定度高，但其他支撐面向薄弱
- 儲蓄不厚、心理壓力沉重，或幾乎沒有可以依靠的支持網絡
- 表面穩定、底層支撐薄弱

**優勢：**
- 具備維持基本生活的經濟結構
- 短期內仍有運作能力，不易立即崩潰

**風險：**
- 安全邊際極小，抗風險能力不足
- 心理或社會支持缺口大時，容易在突發事件下快速滑落

---

### 4. 🐂 努力負重的牛（高負荷撐持型）

**判斷條件（需同時滿足）：**
```
收入穩定度 為 🟢（75–100）
儲備應變力 為 🟠（40–59）
心理與規劃 為 🟡（60–74）
資源連結 為 🟠 或 🔴（0–59）
```

**邏輯說明：**
- 家庭運作穩定、責任清楚，外表看起來很能撐
- 但儲備不算充足，支援網絡有限
- 所有重量幾乎都壓在自己身上
- 穩定但很重，彈性正在慢慢被消耗

**優勢：**
- 穩定度高、責任感強，能長期維持家庭運作
- 心理尚能撐住，具備調整與持續前行的能力

**風險：**
- 長期負荷集中於個人，缺乏支持與緩衝
- 若未補強儲備與資源，突發事件可能造成結構性斷裂

---

### 5. 🐪 撐很久的駱駝（慢性全面吃力型）

**判斷條件（需同時滿足）：**
```
六大指標皆為 🟠（40–59）或其中最多一項為 🟡（60–74）
六大指標中無任何 🔴（0–39）
亦未達烏龜型（多項穩定綠燈）標準
```

**邏輯說明：**
- 收入、儲蓄、保障、管理、心理與支持網絡都處在「還撐得住，但很吃力」的狀態
- 沒有立即的危機，但也沒有真正的綠洲
- 長期處在慢性耗損中

**優勢：**
- 耐力高、適應力強，能在資源有限下長期維持運作
- 面對壓力時不易立即崩解

**風險：**
- 長期慢性壓力累積，易出現身心與財務的同步耗竭
- 若缺乏補給與轉換機制，容易在看似穩定中突然失速

---

### 6. 🦭 被照顧著的水獺（依賴型安全網）

**判斷條件（需同時滿足）：**
```
資源連結 為 🟢（75–100）
心理與規劃 為 🟡（60–74）或 🟢（75–100）
收入穩定度、儲備應變力、金錢管理中至少兩項為 🟠 或 🔴（0–59）
```

**邏輯說明：**
- 身邊有親友、制度或補助資源支持，讓生活不至於沉沒
- 當遇到困難時，總有人或系統能接住
- 內在的安全感很大一部分來自外界的照顧與支撐
- 自我結構尚不足以獨自游得很遠

**優勢：**
- 支持網絡豐富，不易完全陷入孤立或斷援狀態
- 對外部協助的接受度高，較不排斥求助

**風險：**
- 若長期停留在被支撐狀態，自立能力與財務韌性成長緩慢
- 當支持系統變動時，容易出現安全感快速下滑

---

### 7. 🐒 在樹間移動的猴子（社會韌性型）

**判斷條件（需同時滿足）：**
```
資源連結 為 🟢（75–100）
心理與規劃 為 🟢（75–100）
且以下至少一項為 🟠 或 🔴（0–59）：
  收入穩定度
  金錢管理
  儲備應變力
```

**邏輯說明：**
- 經濟結構與金錢能力尚未完全穩定
- 但支持系統強、心理信心高
- 懂得換一條路、找新的支點，而不是原地硬撐
- 韌性來自人際網絡與面對變動時的調整能力

**優勢：**
- 支持網絡與心理調節力強，遇到困難較不易孤立
- 對變動的適應力高，願意嘗試不同解方

**風險：**
- 若長期未補強財務結構與管理能力，可能形成「靠關係撐」而非「靠結構穩」的狀態
- 當支持資源同時受限時，抗風險力仍不足

---

### 8. 🐿️ 躲在葉子下的小松鼠（保護網型受傷狀態）

**判斷條件（需同時滿足）：**
```
資源連結 為 🟢（75–100）
債務與保障 為 🟢（75–100）
且以下任一為 🟠（40–59）：
  儲備應變力
  心理與規劃
```

**邏輯說明：**
- 平時已準備好保險、制度性資源或支持網絡
- 但最近可能遭遇突發支出、健康事件或情緒衝擊
- 讓內心或儲蓄暫時受傷
- 正在恢復與修補中

**優勢：**
- 制度性保障與支持系統完整，不易完全失去安全網
- 願意使用資源與求助，具備修復條件

**風險：**
- 若長期停留在防禦與療傷狀態，可能延後重建行動與結構調整
- 過度依賴保護網，忽略儲備與能力的再累積

---

### 9. 🐻 慢慢探出頭的小熊（恢復中狀態）

**判斷條件（需同時滿足）：**
```
心理與規劃 為 🟢（75–100）
資源連結 為 🟢（75–100）
且以下至少一項為 🟠（40–59）：
  儲備應變力
  金錢管理
  債務與保障
```

**邏輯說明：**
- 曾經經歷過困難的時期，現在內心已逐漸回暖
- 重新對生活產生信心
- 身邊有可以支持的人與資源，心理上也開始看見方向
- 只是部分財務結構仍在修復中
- 心已準備好，結構還在補

**優勢：**
- 心理動能與支持系統穩定，具備實際重建的條件
- 願意面對問題並開始修復，復原力高

**風險：**
- 若修復行動停滯，可能長期停留在準備狀態，無法真正累積結構性穩定
- 過度樂觀而低估財務結構修補所需時間

---

### 10. 🐶 準備出發的小狗（心理啟動型）

**判斷條件（需同時滿足）：**
```
心理與規劃 為 🟢（75–100）
儲備應變力 為 🟠 或 🔴（0–59）
金錢管理 為 🟠 或 🔴（0–59）
不要求資源連結為高
```

**邏輯說明：**
- 已經意識到需要改變，也開始對未來產生期待
- 心裡出現「想試試看、想重新來過」的動力
- 但在儲蓄、預算或理財方法上還沒有清楚的工具與步驟
- 內心充滿動能，但還需要項圈與方向

**優勢：**
- 改變動機與信心已啟動
- 對新方法、新學習的接受度高

**風險：**
- 若缺乏具體工具與支持，行動可能停留在想法與熱情階段
- 過快嘗試高風險決策，反而增加挫折感

---

### 11. 🦅 盤旋高空的老鷹（高能力、低安全網型）

**判斷條件（需同時滿足）：**
```
心理與規劃 為 🟢（75–100）
金錢管理 為 🟢（75–100）
且以下至少一項為 🟠 或 🔴（0–59）：
  儲備應變力
  債務與保障
  資源連結
```

**邏輯說明：**
- 看得遠、判斷清楚，具備良好的分析與規劃能力
- 也懂得如何管理金錢與風險
- 但地面上的安全網仍不夠厚實
- 儲備不足、保障不完整，或可動用的支持資源有限
- 能力很強，但安全邊際仍薄

**優勢：**
- 理解力高、規劃能力佳，學習與調整速度快
- 面對問題時能理性分析並主動尋找解方

**風險：**
- 過度仰賴個人能力，忽略建立制度性保障與儲備
- 一次重大事件可能造成快速且劇烈的下滑

---

### 12. 🐢 穩定前行的烏龜（隱性韌性型）

**判斷條件（需同時滿足）：**
```
六大指標中沒有任何一項為 🔴（0–39）
至少四項指標達 🟡（60–100）
核心三項（收入穩定度、儲備應變力、心理與規劃）皆 ≥ 🟡（60）
```

**邏輯說明：**
- 當未符合 1–11 類時，且同時符合上述條件
- 收入、儲蓄、風險準備、金錢管理、支持網絡與心理狀態都維持在安全區間
- 即使遇到波動，也有足夠的緩衝與調節能力
- 結構完整、抗風險力佳

**優勢：**
- 結構穩定、抗風險能力高，具備長期調整與承受變動的彈性
- 心理穩定、資源與能力配置均衡

**風險：**
- 可能低估外部環境變化速度
- 若過於保守而缺乏成長與更新規劃，容易錯失強化資源與能力的機會

## 五、重要注意事項

### 1. 判斷優先順序
- 判斷順序很重要，因為使用 `if-else` 結構
- 一旦符合某個條件，立即返回該類型，不再繼續判斷
- 優先順序反映風險程度和狀態特徵的明顯程度
- 從最高風險（貓）到最高韌性（烏龜）

### 2. 分數門檻說明
- **🔴 0–39**：高度脆弱（紅燈）
- **🟠 40–59**：結構吃力（橘燈）
- **🟡 60–74**：接近穩定（黃燈）
- **🟢 75–100**：韌性良好（綠燈）

### 3. 邊界型說明
- **駱駝**：慢性全面吃力型
- **水獺**：依賴型安全網
- **老鷹**：高能力、低安全網型
- 邊界型表示處於兩種狀態之間的過渡類型

### 4. 邏輯運算符
- **`&&`（且）**：所有條件需同時滿足
- **`||`（或）**：至少一個條件滿足即可

### 5. 輔助判斷函數
系統使用以下輔助函數來判斷分數區間：
- `isRed(score)`: 判斷是否為 🔴（0–39）
- `isOrange(score)`: 判斷是否為 🟠（40–59）
- `isYellow(score)`: 判斷是否為 🟡（60–74）
- `isGreen(score)`: 判斷是否為 🟢（75–100）
- `isOrangeOrRed(score)`: 判斷是否為 🟠 或 🔴（0–59）
- `isYellowOrGreen(score)`: 判斷是否為 🟡 或 🟢（60–100）

## 六、範例計算

### 範例 1：判斷為「貓」（情況一）
```
答案：{1: 5, 2: 5, 3: 2, 4: 3, 5: 2, 6: 3, 7: 4, 8: 4, 9: 3, 10: 3}

構面分數：
- 收入穩定度 = ((5+5)/2) × 10 = 50
- 儲備應變力 = ((2+2)/2) × 10 = 20 (🔴) ✓
- 債務與保障 = ((3+3)/2) × 10 = 30 (🔴) ✓
- 金錢管理 = ((4+4)/2) × 10 = 40
- 資源連結 = 3 × 10 = 30 (🔴) ✓
- 心理與規劃 = 3 × 10 = 30 (🔴) ✓

結果：符合「貓」的情況一（儲備應變力為 🔴，且債務與保障為 🔴），返回 "cat"
```

### 範例 2：判斷為「螞蟻」
```
答案：{1: 8, 2: 8, 3: 2, 4: 5, 5: 2, 6: 5, 7: 5, 8: 5, 9: 4, 10: 4}

構面分數：
- 收入穩定度 = ((8+8)/2) × 10 = 80 (🟢) ✓
- 儲備應變力 = ((2+2)/2) × 10 = 20 (🔴) ✓
- 債務與保障 = ((5+5)/2) × 10 = 50
- 金錢管理 = ((5+5)/2) × 10 = 50
- 資源連結 = 4 × 10 = 40 (🟠) ✓
- 心理與規劃 = 4 × 10 = 40 (🟠) ✓

結果：符合「螞蟻」的條件，返回 "ant"
```

### 範例 3：判斷為「烏龜」
```
答案：{1: 6, 2: 6, 3: 6, 4: 6, 5: 6, 6: 6, 7: 6, 8: 6, 9: 6, 10: 6}

構面分數：
- 收入穩定度 = ((6+6)/2) × 10 = 60 (🟡) ✓
- 儲備應變力 = ((6+6)/2) × 10 = 60 (🟡) ✓
- 債務與保障 = ((6+6)/2) × 10 = 60 (🟡) ✓
- 金錢管理 = ((6+6)/2) × 10 = 60 (🟡) ✓
- 資源連結 = 6 × 10 = 60 (🟡) ✓
- 心理與規劃 = 6 × 10 = 60 (🟡) ✓

結果：
- 無任何 🔴 ✓
- 至少四項達 🟡（實際六項都達 🟡）✓
- 核心三項皆 ≥ 🟡 ✓
- 返回 "turtle"
```
